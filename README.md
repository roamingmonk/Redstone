# Terror in Redstone

A professional-grade 2D RPG framework, evolving from a Pygame prototype into a modular, event-driven architecture.  
Think of it as a tavern tale turned software engineering exercise: branching dialogues, JSON-driven content, and a future-proof design.

---

## Quick Start

```bash
# clone repo
git clone https://github.com/USERNAME/terror-in-redstone.git
cd terror-in-redstone

# set up environment
python3 -m venv .venv
source .venv/bin/activate   # Windows: .venv\Scripts\activate

pip install pygame

# run the game
python main.py
